# Lawnkeeperapp

To start your Phoenix server with docker:

```
git clone https://github.com/thelawnforum/lawnkeeperapp.git
cd lawnkeeperapp
docker-compose up
```

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
